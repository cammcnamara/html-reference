Extreme Nested Div Stress Test

This is a test to see how the browser handles a large number of deeply nested <div> elements.

Hypothesis: The browser will crash

Results: The browser does crash giving error code 11. Upon reasearch I have found this means the browser has run out of memory and has been killed by the OS.
